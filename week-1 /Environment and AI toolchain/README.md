# Frontend AI Capstone Project

A React and Tailwind CSS capstone application built as part of a frontend AI development course. This repository will hold the final project demonstrating modern frontend development practices with AI-assisted workflows.

## Tech Stack

- **React** — UI library
- **Tailwind CSS** — utility-first styling
- **npm** — package management and build scripts

## Project Structure

```
frontend-ai-capstone/
├── README.md       # Project overview and setup
├── CLAUDE.md       # AI assistant rules (build commands, stack)
├── LICENSE         # MIT license
└── .gitignore      # Ignored paths (e.g. node_modules/)
```

As the app is built out, source code will typically live under `src/` with entry points such as `index.html` and `package.json` at the project root.

## Getting Started

Once dependencies are added:

```bash
npm install
npm run build
```
